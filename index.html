a few me

projects

Boston Housing Price Prediction

A project concerning the prediction of housing prices, which I carried out for the purposes of one of the tasks at the Polish-Japanese Academy of Information Technology. My goal was to create an algorithm based on the prediction of home prices in Boston, so as to invest money well. The project was written in Python and made in Jupyter Notebook. The main libraries I used in this project are pandas, numpy, sklearn, seaborn, matplotlib, xgboost and many more. The data I used for this project comes from Kaggle. The built-in algorithm was based on linear regression, lasso and xgboost models.

Assignment of users in the dating app

The project concerns the creation of a Machine Learning model, which I carried out for the needs of one of the tasks at the Polish-Japanese Academy of Information Technology. The aim of this project was to create an algorithm based on the appropriate number of clusters and assign users of the dating application to these clusters. The data that was used was taken from Kaggle. To build this model, I used libraries such as pandas, numpy, sklearn, seaborn, matplotlib. Thanks to that, introducing this algorithm to the application will enable more accurate selection of people, thanks to which the application will work better and users will spend more time on it.

Sea disaster analysis based on the Titanic

This is my first project that I carried out for one of the tasks at the Polish-Japanese Academy of Information Technology. This project was based on data analysis and drawing conclusions regarding and answering the question of how to increase safety in sea expeditions. I created a Machine Learning model based on a decision tree that learned from training data and assigned labels so that it could predict new events. I used libraries such as pandas, numpy, sklearn in it.

Data analysis relating to emergency calls

The project concerns the analysis of emergency telephone data in Montgomery County, Pennsylvania, USA. The data used was taken from Kaggle and includes data from 2015. The main aim of the analysis was to present a heat map from which many conclusions can be drawn to minimize the number of such reports in the future. I believe the analysis has identified several issues that may help increase security. The possibilities offered by data analysis in Python are huge. To carry out this analysis, I used libraries such as pandas, numpy, seaborn, matplotlib.

Automated algorithm

The project was carried out for one of the tasks at the Warsaw University of Technology. My goal was to create a tool that would be as optimized as possible, speed up the calculation process and give the opportunity to make changes at any time. The sheet I created is used to calculate the entire project. I made this project in Microsoft Excel, I did it on my own initiative to facilitate the calculations. During the implementation of this project, I paid special attention to the fact that the supported spreadsheet in Microsoft Excel was as clear and easy to use as possible.

Autonomous data collection

As part of one of the projects carried out at the Polish-Japanese Academy of Information Technology, I wrote an algorithm in R. This algorithm was designed to autonomously collect information about audi cars on otomoto.pl and save this data in the SQL Server relational database.

Website as a portfolio

The project is about creating a website that aims to present me, main goals, interests and technology that I use every day, and some of my projects. Technologies that I used to make this website are HTML and CSS and JavaScript which allowed me to make animations.

Building processes using SQL and relational databases

On a daily basis, they participate in a process project aimed at combining mobile and fix data as one process. The technologies it uses for this are SQL, Teradata, Hive and Oracle. I also write reports in SQL Server and fix errors that appear.
